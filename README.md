📊 Advertising Sales Prediction ML Web App

📌 Project Overview

This project is a Machine Learning web application that predicts product sales based on advertising budgets across different media channels such as TV, Radio, and Newspaper. The application allows users to input advertising budgets and get predicted sales using multiple regression models.

The project demonstrates the complete Machine Learning workflow including data preprocessing, model training, model comparison, and deployment using Streamlit.

---

🚀 Features

- Predict sales using advertising budget
- Multiple Machine Learning models:
  - Linear Regression
  - Ridge Regression
  - Lasso Regression
- Model comparison
- Interactive Streamlit web interface
- Visualization of predictions
- End-to-end ML project

---

🧠 Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Data Scaling
5. Model Training
6. Model Evaluation

▶️ How to Run the Project

1. Clone Repository

git clone https://github.com/anujd1432/advertising-sales-ml-app.git

2. Install Requirements

pip install -r requirements.txt

3. Run Streamlit App

streamlit run app.py
